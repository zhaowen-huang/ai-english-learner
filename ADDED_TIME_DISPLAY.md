# 单词加入时间显示功能

## 更新内容

为每个单词添加了清晰的加入时间显示，让用户可以清楚地看到每个单词是什么时候添加到生词本的。

## 实现细节

### 1. 生词本列表 (`app/(tabs)/vocabulary.tsx`)

**改进前：**
- 时间显示在底部，格式简单（如 "Apr 14"）
- 没有明显的视觉标识

**改进后：**
- 使用徽章样式显示时间
- 添加日历图标 📅 作为视觉标识
- 完整的日期格式（YYYY/MM/DD）
- 浅灰色背景突出显示
- 始终显示在卡片右下角

**视觉效果：**
```
┌─────────────────────────────┐
│ hello 🔉 [已掌握]      🗑️  │
│                             │
│ "Hello, how are you?"       │
│                             │
│ ─────────────────────────── │
│ 📄 example.com    📅 2026/04/14 │
└─────────────────────────────┘
```

### 2. 单词详情页 (`app/word/[word].tsx`)

**改进前：**
- 只有在有文章来源时才显示时间
- 时间显示不够突出

**改进后：**
- 无论是否有文章来源，都显示加入时间
- 使用独立的徽章样式
- 添加 "加入于" 文字说明
- 完整的日期格式（2026年4月14日）
- 浅灰色背景，更加醒目

**视觉效果：**
```
┌─────────────────────────────┐
│ ← 返回                      │
│                             │
│ hello 🔊                    │
│ 你好                        │
│                             │
│ ─────────────────────────── │
│                             │
│ 上下文                      │
│ | "Hello, how are you?"     │
│                             │
│ 来源                        │
│ example.com                 │
│                             │
│ 📅 加入于 2026年4月14日      │
│                             │
│ 学习进度                    │
│ ...                         │
└─────────────────────────────┘
```

## 样式特点

### 生词本列表
- **背景色**: 浅灰色 (`colors.neutral[100]`)
- **圆角**: 小圆角 (`borderRadius.sm`)
- **内边距**: 紧凑布局
- **图标**: 📅 emoji
- **字体**: 加粗 (fontWeight: '500')
- **颜色**: 次要文本色

### 单词详情页
- **背景色**: `#F5F3F0` (温暖的浅灰色)
- **圆角**: 6px
- **内边距**: 水平 12px，垂直 8px
- **图标**: 📅 emoji (14px)
- **文字**: "加入于" + 完整日期
- **颜色**: `#8B8680` (柔和的灰色)

## 技术实现

### 日期格式化

使用 JavaScript 原生 `toLocaleDateString` API：

```typescript
// 生词本列表 - 简洁格式
new Date(vocab.createdAt).toLocaleDateString('zh-CN', {
  year: 'numeric',
  month: '2-digit',
  day: '2-digit'
})
// 输出: 2026/04/14

// 单词详情页 - 完整格式
new Date(vocab.createdAt).toLocaleDateString('zh-CN', {
  year: 'numeric',
  month: 'long',
  day: 'numeric'
})
// 输出: 2026年4月14日
```

### 布局优化

使用 Flexbox 确保时间徽章始终在右侧：

```typescript
<View style={styles.cardFooter}>
  <View style={styles.footerLeft}>
    {/* 文章来源 */}
  </View>
  <View style={styles.footerRight}>
    {/* 时间徽章 */}
  </View>
</View>
```

## 用户体验提升

1. **可见性**: 时间信息更加突出，不会被忽略
2. **一致性**: 所有单词都显示加入时间
3. **可读性**: 使用中文日期格式，更符合习惯
4. **美观性**: 徽章样式与整体设计协调
5. **信息完整**: 用户可以追溯单词的学习历史
